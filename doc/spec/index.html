Before Getting Started

This project, Intel® RealSense™ technology with JavaScript API for Linux OS works on Intel® RealSense™ SDK for Linux.

Before getting started, read the introduction to get to know the concepts, basic principles and then install Intel® RealSense™ SDK for Linux on to your computer.

Specifications to Person Module, Object Module and SLAM Module

Person Module
This specification describes the API of Person tracking, localization, and recognition capabilities working on Intel® RealSense™ camera.
Object Module
This specification describes the API of Object recognition, localization, and tracking capabilities working on Intel® RealSense™ camera.
Simultaneous Localization and Mapping (SLAM) module
This specification describes the API of SLAM module working on Intel® RealSense™ camera.
Common API in all modules
This specification describes the common API resides in all modules.
